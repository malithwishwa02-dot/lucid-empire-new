# Lucid Empire (Research Copy)

This is a minimal, research-friendly copy of the Lucid Empire repository. All binaries, large assets, and cache files have been removed. Only source code, scripts, configs, and essential documentation are included.

## Structure
- All Python source code, patches, and scripts are preserved.
- No compiled binaries, browser builds, or large font/assets are present.
- This copy is for code review, research, and analysis only.

## Excluded
- bin/, bundle/, fonts/, fontconfigs/, __pycache__, *.o, *.zip, *.png, *.pdf, and all large binary files.

## Included
- All .py, .sh, .md, .txt, .yml, .json, .cfg, .patch, and essential folders (core, modules, network, scripts, etc.).

## Usage
This copy is not runnable as a browser. For full builds, use the original repository.
